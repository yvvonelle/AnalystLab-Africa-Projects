# AnalystLab Africa Internship 
## Week 1–2 Task: 

This repository contains my submission for the AnalystLab Africa 
Data Analytics Internship Program Tasks for week 1 and 2. The task involved 
cleaning, validating, and exploring two real-world datasets using 
Python in Jupyter Notebooks.

---

## Datasets
1. **E-commerce Transactions Dataset** — Online retail transaction 
   records from a UK-based store covering December 2010 to December 2011
2. **Netflix Movies & TV Shows Dataset** — Netflix content library 
   containing movies and TV shows up to 2021

---

## Tools & Libraries
- Python 3
- Pandas — data manipulation and cleaning
- Matplotlib — data visualization
- Seaborn — statistical visualizations
- Jupyter Notebooks — development environment
- VSCode — code editor

---

## Task Structure
Each dataset follows the same 5-task structure:

### Task 1 — Dataset Understanding
- Loaded each dataset into Pandas
- Identified shape, data types, numerical and categorical columns
- Identified missing values and possible primary keys
- Described what each dataset contains

### Task 2 — Data Cleaning
- Identified and handled missing values
- Removed duplicate records
- Standardized column names, text casing and date formats
- Validated data by removing invalid, inconsistent and incorrect records
- Documented every cleaning step with a summary table

### Task 3 — Exploratory Data Analysis
- Generated summary statistics for numerical columns
- Performed 5+ exploratory analyses per dataset including:
  - E-commerce: top products, revenue by country, monthly trends,
    top customers, orders by country
  - Netflix: content type distribution, content added by year,
    top producing countries, common ratings, common genres

### Task 4 — Visualizations
- Created 7 visualizations per dataset including:
  - Bar charts, line charts, pie charts, histograms and box plots
- Each visualization includes a title, axis labels and key finding

### Task 5 — Insights
- Identified 5 key insights per dataset
- Each insight is supported by analysis and referenced visualizations

---

## Key Findings

### E-commerce Dataset
- Started with 541,909 rows, cleaned down to 524,878 rows
- United Kingdom generates over 90% of total revenue at £9 million
- Revenue peaks in November 2021 due to holiday shopping season
- A small group of wholesale customers drive a large portion of revenue
- Most products are priced below £5 confirming a low-cost retail model

### Netflix Dataset
- Started with 8,807 rows, cleaned down to 8,790 rows
- Movies make up 69.7% of content vs 30.3% TV Shows
- Netflix content grew massively from 2016 peaking in 2019
- United States and India are the top content producing countries
- TV-MA is the most common rating confirming adult focused content
- International Movies is the top genre showing Netflix's global strategy
